FUNCIONAMIENTO
Para compilar basta con ejecutar el comando "make all" desde una terminal de Linux (Bash), el cual crea el ejecutable "programa", el cual se ejecuta
de la siguiente manera ./programa,donde el archivo a compilar debe llamarse "main.c"


ESPECIFICACIONES Y ACLARACIONES
