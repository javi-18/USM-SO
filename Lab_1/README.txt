FUNCIONAMIENTO
Para compilar basta con ejecutar el comando "make all" desde una terminal de Linux (Bash), el cual crea el ejecutable "programa", el cual se ejecuta
de la siguiente manera ./programa,donde el archivo a compilar debe llamarse "main.c"(esto es para probar tanto el codigo optimizado como el no optimizado)



ESPECIFICACIONES 
Para que el programa funcione todos los archivos de texto que se vayan a probar deben estar dentro del mismo directorio donde se va a correr
el programa, los cuales deben contener la orientacion en el formato "horizontal" o "vertical"
Si se quiere volver a correr el programa hay que borrar manualmente las carpetas y tambien mover manualmente los archivos para volver a correr el programa, el cual debe llamarse "main.c"
para  que de esa manera se puedan comparar los tiempos de ejecucion entre ambos codigos
